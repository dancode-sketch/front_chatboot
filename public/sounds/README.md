Coloca aquí los archivos de audio que quieres reproducir con las notificaciones.

Ejemplos de nombres usados por defecto en el código:

- new-order.mp3 (pedido nuevo)
- new-message.mp3 (mensaje entrante)
- status-change.mp3 (cambio de estado de pedido)

Puedes usar cualquier formato compatible con HTMLAudioElement (mp3, wav, ogg...).

Si el navegador bloquea la reproducción automática, normalmente se requiere que el usuario interactúe con la página primero (clic, toque, etc.).
